# vueElement

> vueMobile by eastmoney.com

## 安装运行导引

``` bash
# 安装依赖包
npm install

# 启动服务，支持热加载
npm run dev

# 生产环境，工程化打包压缩
npm run build

# build for production and view the bundle analyzer report
npm run build --report


├── src/·····················<- 组件开发目录
│   ├── assets/··············<- 静态资源文件
│   ├── components/··········<- 开发依赖的组件
│   ├── filter/··············<- 常用过滤器
│   ├── routes/··············<- 页面路由配置
│   ├── scss/················<- 样式文件，主要是公用的样式文件
│   ├── ├── components/······<- 组件样式
│   ├── ├── views/···········<- 测试组件样式
│   ├── view/················<- 组件测试用例   
│   ├── utils/···············<- 常用工具类方法     
│   ├── App.vue··············<- 入口文件      
│   ├── main.js··············<- 入口配置文件

```
